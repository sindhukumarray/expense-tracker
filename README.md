# Expense Tracker

A modern and responsive Expense Tracker web application built using HTML, CSS, and JavaScript.
This project helps users manage their income and expenses with a clean dashboard UI and Local Storage support.

---

##  Live Demo

🔗 [https://sindhukumarray.github.io/expense-tracker/](https://sindhukumarray.github.io/expense-tracker/)

---

##  Features

* Responsive modern dashboard UI
* Add income and expense transactions
* Separate income and expense transaction sections
* Dynamic balance calculation
* Total income and total expense summary
* Delete individual transactions
* Clear all transactions functionality
* Local Storage support for data persistence
* Mobile responsive design
* Modern card layout and hover effects
* Date & time support for transactions
* Empty state messages when no transactions exist

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Remix Icons

## How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/sindhukumarray/expense-tracker.git
```

2. Open the project folder

```bash
cd expense-tracker
```

3. Open `index.html` in your browser
